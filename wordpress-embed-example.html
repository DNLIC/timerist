WordPress Embed Examples

Copy and paste these code snippets into your WordPress pages or posts.

1. Countdown Timer

<iframe src="https://yourdomain.com/timerist/countdown.html" 
        width="100%" 
        height="600" 
        frameborder="0" 
        scrolling="no">
</iframe>

Instructions

  1. Replace https://yourdomain.com with your actual domain where Timerist is hosted
  2. In WordPress, add a "Custom HTML" block to your page or post
  3. Paste the iframe code for the timer you want to embed
  4. Adjust the height value if needed (600px works well for most timers)
  5. Publish your page!

Note: Make sure your WordPress site allows iframe embeds. Some security plugins may block iframes by default.
