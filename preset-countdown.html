Preset Countdown Help

Preset Countdown is for quick fixed-duration timers (rests, holds, bursts).

Quick start
Tap any preset (15s, 30s, 45s, 1:00, etc.) to start instantly.

Custom workout
Tap Set Up (or double-tap the timer display) to open the settings panel to set any custom time or change the completion message.

During the session
Pause, Reset, or Clear as needed. The display shows large countdown numbers.

iPhone / iPad
Tap Enable Sound in the banner.
